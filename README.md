# dto-example

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

This project was created using `bun init` in bun v1.3.0. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.


## Requerimentos de validacion

- nombre entre 3 y 20 caracteres
- tipo no nullo  que sea uno de los siguientes: "CANDIED_FRUIT" | "CHOCOLATE_CHIPS" | "PLAIN"
- peso entre 100 y 1000 gramos
- marca (brand) string no nullo
- tier no nullo que sea uno de los siguientes: "LOW_COST" | "STANDARD" | "PREMIUM"
